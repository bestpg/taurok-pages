Über das Projekt

taurok.com ist meine eigene Portfolio-Website unter einer selbst registrierten Domain. Gebaut habe ich sie gemeinsam mit dem KI-Agenten Claude: Ich gebe Idee und Design vor und setze sie Schritt für Schritt mit der KI um.

Die Seite besteht aus reinem HTML und CSS und ist über die eigene Domain live im Internet veröffentlicht. Dabei habe ich praktisch gelernt, wie man eine Domain einrichtet, eine Website hostet und sie online bringt.

Eckdaten

  • Eigene Domain: taurok.com
  • Gemeinsam mit dem KI-Agenten Claude gebaut
  • Reines HTML & CSS, ohne Framework
  • Live im Internet veröffentlicht

Für mich war taurok.com ein wichtiger Schritt, weil ich zum ersten Mal eine eigene Website komplett selbst online gebracht habe.
